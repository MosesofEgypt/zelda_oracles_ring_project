This project aims to overhaul the rings system in Oracle of Ages and Oracle of Seasons through targeted ASM patches.

This project would not have been possible without having the oracles-diasm project for reference.
If ZORP interests you, please check out the reference project here:
    https://github.com/Stewmath/oracles-disasm/tree/master

Many features of this project include:
    * All rings carried can be equipped at once
    * Ring box sizes can be changed from 0 to 10 rings
    * Wallet size increased from 999 to 9999
    * Gasha seed price reduced from 300 rupees to 50
    * Transform rings don't prevent item usage(all actions can be performed while wearing one)
    * Every ring does something(trophy rings change your color, advance rings give you more invincibility frames, etc)
    * All rings have been reworked to be useful in some way(no more niche/barely useful rings)
    * Rings that do similar things(i.e. power/armor/joy/etc rings) can stack
    * Ability to punch with fist/experts ring with only 1 hand free
    * Several bad rings have been replaced with a handful of custom ones
    * Ability to swap rings at any time(depends on the level of your ring box)
    * Hidden ring-combo effects, which occur when specific rings are worn together
